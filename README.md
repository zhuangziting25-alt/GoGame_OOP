## 2. 程式介紹
本專題使用 C++ 開發，落實物件導向中的**封裝**與**模組化**設計。

### UML 類別圖
以下為本系統的架構圖：



* **Board 類別**：負責管理 $19 \times 19$ 的棋盤資料（使用 `std::vector` 封裝），提供落子與顯示功能。
* **GameManager 類別**：負責遊戲邏輯控制，包含玩家輪替、輸入處理與終止判定。

## 3. 安裝與執行教學
1.  下載本專案原始碼。
2.  確保環境已安裝 C++ 編譯器 (如 GCC)。
3.  編譯指令：`g++ main.cpp Board.cpp GameManager.cpp -o GoGame`
4.  執行指令：`./GoGame` (Windows 則執行 `GoGame.exe`)

## 4. 成果展示
* **棋盤顯示**：使用 `.` 代表空位，`B` 代表黑棋，`W` 代表白棋。
* **操作方式**：玩家輸入座標 (x y) 進行落子。
